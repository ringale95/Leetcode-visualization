Array Indexing

What is an array?

An array is a contiguous set of memory locations containing the same datatype for a specified number of locations.

Array indexing refers to the process of accessing individual elements within an array by their position or index.

How to initialize an array in Java?

int[] arr = new int[5];
int[] myArray = {1, 2, 3, 4, 5};

Java code to access the value at index

There are different ways:

  • Direct access by Index: myArray[0]
  • For loop:
    for(int i=0; i < myArray.length; i++)
    System.out.println(myArray[i]);
  • For:each loop
    int index = 0; for(int number : myArray) System.out.println("Element at index" + index + ":" +number);
