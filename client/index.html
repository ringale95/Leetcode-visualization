What is a Multi-dimensional array?

Multi-dimensional arrays in Java are arrays that contain arrays as their elements, allowing for more complex data structures such as matrices or tables

The most common type of multi-dimensional array is the two-dimensional array, but arrays of three or more dimensions can also be created.

How to access multi-dimensional array?

We use two indices: one for row and other for column
int element = matrix[1][2]; //Accessing
matrix[2][3] = 13; //Modifying

How to declare and initialize multi-dimensional array?

Declaring 2-D array: matrix = new int[3][3];
int[][] matrix = {
{1,2,3},
{4,5,6},
{7,8,9} };

How to iterate through multi-dimensional array?

We can use nested loop to iterate through each of the element

for(int i = 0; i < matrix.length; i++){
for (int j = 0; j < matrix[i].length; j++){
System.out.println("Element at [" + i + "][" + j + "] = " + matrix[i][j]);
} }
